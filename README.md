# ✈️ Assistente de Viagem com IA

> Assistente de viagem com IA generativa e RAG que planeja roteiros, responde dúvidas, recomenda atrações e ajuda o turista em tempo real, 24/7 e multilíngue.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/turismo-chatbot-viagem/

## 📊 Resultados-chave (KPIs)
**88%** Resolução · **12** Idiomas · **24/7** Disponível · **4.7/5** CSAT

## 🧠 Tecnologias de IA
GenAI · RAG · LLM

## ✨ O que faz
- Planejamento de roteiro
- Recomendação de atrações

- Multilíngue (12 idiomas)
- Assistência 24/7
